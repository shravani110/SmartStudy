# Smart Study Quiz Generator - App Overview

## What is this app?

An Android application that generates study quizzes from text using AI technology. Users can paste study material and get instant quiz questions with answers.

## Key Features

- Generate quiz questions from any text
- AI-powered question creation using Gemini API
- Clean, easy-to-use interface
- Bottom navigation for different sections
- Stores quizzes locally on device

## How to Use

1. Open the app
2. Enter or paste your study material
3. Tap "Generate Quiz"
4. Review questions and answers
5. Save for future reference

## API Connection

The app connects to Google's Gemini AI service for question generation. Configuration is in the `.env` file.

## Installation

1. Install the APK file on your Android device
2. Open the app - no additional setup required
3. Start using immediately

## Requirements

- Android device (version 7.0 or higher recommended)
- Internet connection for quiz generation
- Approx 50MB free storage space

## Troubleshooting

If quizzes don't generate:
- Check internet connection
- Ensure API key is valid
- Try with shorter text first
